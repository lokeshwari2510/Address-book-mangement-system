# Address Book Project

A simple **Address Book Management System** developed in **C programming**.  
This project allows users to store, search, edit, and manage contact details efficiently using file handling concepts.

---

## Features

- Add new contacts
- Search contacts by name or phone number
- Edit existing contact details
- Delete contacts
- Display all contacts
- Save contacts permanently using file handling
- User-friendly menu driven program

---

## Technologies Used

- C Programming
- Structures
- File Handling
- Functions
- Arrays and Strings

## How to Compile and Run

### Compile

```bash
gcc *.c -o addressbook
```

### Run

```bash
./addressbook
```
